To-Do List API

A simple API for managing a list of to-do items.

API Reference

GET /items

Returns a list of all to-do items in the system.

Parameters

[]

Responses

{
  "200": {
    "description": "A list of items was successfully retrieved.",
    "content": {
      "application/json": {
        "schema": {
          "type": "array",
          "items": {
            "$ref": "#/components/schemas/Item"
          }
        }
      }
    }
  }
}
